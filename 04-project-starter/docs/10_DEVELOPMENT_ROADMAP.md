## Build Priority Rule

Always build the workflow before the automation.

Example:

1. Build editable storyboard table.
2. Then add AI generation into that table.
3. Build SME review fields.
4. Then add AI SME questions.
5. Build QA issue tracker.
6. Then add AI QA checks.

This prevents the app from becoming a fragile AI wrapper.
