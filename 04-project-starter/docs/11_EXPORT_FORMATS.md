# 11 Export Formats

## Word Storyboard Export

Sections:

1. Course information
2. Target audience
3. Course goal
4. Learning objectives
5. Module structure
6. Storyboard table
7. SME review questions
8. Assessment items
9. Accessibility notes
10. QA checklist
11. References

Storyboard table columns:

- screen number
- module
- screen title
- on-screen text
- narration/script
- interaction type
- visual/media direction
- assessment/reflection
- SME review notes
- accessibility notes
- status

## Excel Tracker Export

Sheets:

1. Course overview
2. Storyboard tracker
3. SME review tracker
4. QA issues
5. Assessment bank
6. Media tasks
7. Handoff checklist

## PPT Draft Export

Each storyboard screen becomes one PPT slide.

Slide notes should include:

- narration
- interaction notes
- visual/media direction
- SME review notes
- accessibility notes

The PPT draft is intended for Storyline import or stakeholder preview, not final course delivery.

## SME Review Package

Sections:

1. Review instructions
2. Course summary
3. Items requiring SME approval
4. High-risk content list
5. Storyboard screens for review
6. Targeted SME questions
7. Approval checklist

## QA Checklist Export

Sections:

- instructional design checks
- accessibility checks
- content quality checks
- assessment alignment checks
- media readiness checks
- SME approval checks
- LMS testing readiness checks

## Handoff Package

Final handoff should include:

- final storyboard
- source files
- editable course files
- SCORM package if applicable
- Storyline/Rise/HTML5 files if applicable
- multimedia source files
- audio files
- transcripts/captions
- image assets
- job aids
- quiz bank
- survey/evaluation tools
- SME approval record
- QA checklist
- LMS testing notes
- maintenance schedule
