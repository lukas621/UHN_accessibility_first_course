## Naming Convention

```
voiceover_{screen}_{layer}.mp3
voiceover_{screen}_{layer}.srt
```

Examples:
- `voiceover_1.4_tab2_social.mp3` — Screen 1.4, Tab 2 (Social Model)
- `voiceover_1.7_choiceA.mp3` — Screen 1.7, Choice A feedback
- `voiceover_1.12_card3.mp3` — Screen 1.12, Practice Tip Card 3

## Production Notes

- **Voice:** Warm, professional, conversational. Canadian English pronunciation.
- **Pacing:** 150 words per minute base. Slow slightly for key terms on first introduction.
- **Beats:** Honour all (Beat) markers with 1–2 second pauses. 2–3 seconds before scenario choices.
- **Format:** MP3 (192kbps) + SRT caption files
- **Pronunciation:** "AODA" as "ay-oh-dee-ay." Canadian spelling ("centre," "colour").
