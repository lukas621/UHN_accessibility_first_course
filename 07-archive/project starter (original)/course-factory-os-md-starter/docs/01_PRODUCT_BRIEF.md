# 01 Product Brief: Course Factory OS

## Working Product Name

Course Factory OS

Alternative names:

- LearnOps Factory
- StoryboardOps
- Course Production OS
- Accessibility First Production Hub

## One-Sentence Pitch

Course Factory OS helps instructional designers convert large SME/source materials into structured, SME-reviewable, accessible, build-ready eLearning storyboards and handoff packages.

## Problem

Instructional designers often receive large, messy, or SME-heavy source materials and must manually transform them into usable eLearning deliverables. The process includes analysis, learning objectives, storyboarding, SME review, media planning, accessibility checks, LMS testing, and handoff.

This process is time-consuming because:

- Source materials are long and unstructured.
- SMEs do not always know what IDs need.
- Course objectives, assessments, activities, and media plans are often created manually.
- SME review cycles are messy and inconsistent.
- Accessibility and QA checks happen too late.
- Handoff packages are inconsistent.
- Multi-course series require strong consistency across lessons.

## Target User

Primary user:

- Instructional designer working on healthcare, compliance, accessibility, onboarding, or regulated training.

Secondary users later:

- SME
- course owner
- L&D manager
- multimedia developer
- QA reviewer
- LMS admin

## First Use Case

A large accessibility-related course series with 18 guides. The document is approximately 400+ pages and includes repeated guide structures such as:

- purpose
- primary focus
- learning emphasis
- core objectives
- key themes
- scenarios
- reflection questions
- inclusive practice tips
- action planning
- references

This use case is ideal because it proves the app can handle:

- a large source document
- multiple courses
- repeated instructional patterns
- accessibility-specific QA
- SME review
- course-series consistency

## Value Proposition

Turn a large source document into structured course production assets in hours instead of days or weeks.

More specifically:

- Split one large source document into multiple course records.
- Extract objectives, scenarios, reflection questions, tips, references, and key themes.
- Generate course blueprints.
- Generate editable storyboards.
- Suggest interactions, assessments, and performance support tools.
- Route high-risk content to SMEs.
- Run accessibility and instructional design QA.
- Export reusable production packages.

## Product Category

Not a course generator.

Course Factory OS is a **LearnOps / eLearning production workflow system**.

It sits before or beside tools like Storyline, Rise, Brightspace, Canvas, Moodle, Captivate, and HTML5 production workflows.

## MVP Success Criteria

The MVP is successful if one instructional designer can:

1. Upload or paste a large source guide.
2. Generate a course blueprint.
3. Generate a clean storyboard table.
4. Edit the storyboard.
5. Generate SME review questions.
6. Generate assessments and activities.
7. Run QA checks.
8. Export useful Word/Excel/PPT-ready production assets.
9. Track progress across multiple courses.

## Long-Term Vision

Course Factory OS becomes the production control center for ID teams. It standardizes how courses are planned, reviewed, built, tested, published, maintained, and handed off.
